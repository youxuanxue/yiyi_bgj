# 小游戏集合 🎮

这是一个包含多个有趣小游戏的集合仓库。

## 游戏列表

### 👊 [左右手无聊搏击](./left-right-boxing/)
一个轻松的对战小游戏，左右手分别控制拳套收集宝石和水果，在12秒内比拼收集数量！

**特点：**
- 8种不同颜色和款式的拳套武器
- 21种收集物（14种水果 + 7种宝石）
- 绚丽的爆炸特效和音效
- 12秒快节奏对战

[进入游戏](./left-right-boxing/index.html) | [查看说明](./left-right-boxing/README.md)
